# Launch Video Skill

プロダクトローンチ動画スクリプト作成スキル。

## Knowledge Location

`ローンチ動画/` ディレクトリを参照

### 4話構成（各話70分・8パート）

**第1話：導入と問題提起**
- パート1：常識の破壊（0〜7分）
- パート2：ローンチコンセプト（7〜15分）
- パート3：ラポールの形成（15〜25分）
- パート4：問題提起（25〜35分）
- パート5：シークレット1の解禁（35〜45分）
- パート6：実践の条件（45〜55分）
- パート7：次回予告（55〜65分）
- パート8：クロージング+CTA（65〜70分）

**第2話：深化と信頼構築**
- 選民意識の形成
- コア哲学の共有
- シークレット2の解禁
- 悩み解決
- 成功事例の紹介

**第3話：変革と決意**
- 帰属意識の醸成
- 過去との決別
- 自分を超える挑戦
- シークレット3の解禁

**第4話：オファーとクロージング**
- 成長の集大成
- 最終関門の提示
- オファー概要の提示
- 反論つぶし・サポート・保証
- CTA・特典説明
- クロージング

### 3話構成（凝縮版）
- 4話構成を3話に圧縮
- 各パートの凝縮版ナレッジ

## Usage

```
ユーザー: ローンチ動画の台本を作って
→ 4話構成または3話構成を確認して作成

ユーザー: 第1話のスクリプトを書いて
→ 第1話の8パート構造に沿って作成

ユーザー: オファー動画を作りたい
→ 第4話の構造を参照
```

## Guidelines

- 各パートの時間配分を守る
- シークレット解禁で期待を高める
- CTAは各話末に設置
- 次回予告で視聴継続を促す
