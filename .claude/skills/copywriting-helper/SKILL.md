# Copywriting Helper

マーケティング・コピーライティングのナレッジベースを活用したライティング支援スキル。

## Instructions

1. ユーザーの目的を確認（セールスレター、メール、VSL等）
2. プロジェクト内の該当ナレッジを参照
3. 適切な構造・テンプレートを適用
4. ターゲットに合わせたコピーを生成

## Available Knowledge

### ステップメール
- 6つの教育要素（目的、問題点、手段、投資、信用、行動）
- メルマガタイトル・構成の奥義
- 日刊メルマガ執筆ナレッジ

### セールスレター
- ヘッドコピー作成
- ベネフィット提示
- 価格提示・価格根拠
- クロージング
- 追伸パート

### VSL（15章構成）
- 冒頭フック
- Before/Afterストーリー
- 証拠提示
- オファー・クロージング

### ローンチ動画（4話構成）
- 各話70分・8パート構成
- シークレット解禁の流れ
- CTA設計

## Usage Examples

```
ユーザー: 新商品のセールスレターを書いて
→ セールスレター/旧版/ のナレッジを参照し、構造に沿って作成

ユーザー: メルマガの件名を考えて
→ メルマガタイトル奥義を参照し、開封率を高める件名を提案

ユーザー: VSLの台本を作成して
→ 15章構成に沿ってスクリプトを生成
```

## Guidelines

- 商材系コピーライティングの専門用語を適切に使用
- 感情を動かす表現を重視
- 構造とリズムを意識した文章作成
