# NanoBanana プロンプト チートシート

## クイックリファレンス

### 強調テクニック
| 目的 | 方法 | 例 |
|------|------|-----|
| 必須指示 | ALL CAPS | `MUST include`, `NEVER remove` |
| 複数指示 | Markdownリスト | `- item 1\n- item 2` |
| 色指定 | HEX + 名前 | `royal blue (#4169E1)` |
| 品質向上 | 受賞キーワード | `Pulitzer Prize winning` |

## 即使えるプロンプト構文

### 基本形
```
[被写体], [スタイル], [照明], [構図], [品質]
```

### 商品写真
```
Product photo of [商品], white background, soft lighting, centered, commercial quality
```

### ポートレート
```
Portrait of [人物], 85mm lens, Rembrandt lighting, shallow DOF, magazine quality
```

### 風景
```
Landscape of [場所], golden hour, wide angle, foreground interest, National Geographic quality
```

### 編集
```
Edit: [変更内容]. PRESERVE original [保持要素]. MUST [必須条件].
```

## よく使う否定形
```
NO text in image
NEVER include watermarks
WITHOUT people in background
AVOID cluttered composition
```

```
square format (1:1)
```
