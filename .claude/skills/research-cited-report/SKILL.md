# Research: Cited Report

## Instructions

1. 問いを再定義し、前提・範囲・比較軸を明確化する
2. 検索クエリ案を3〜8個作り、一次情報を優先して収集する
3. 主張は出典とセットにし、矛盾点があれば併記する
4. 仕上げに「結論 / 重要ポイント / リスク / 次アクション」をまとめる

## Output format

- Executive summary
- Key findings (with sources)
- Trade-offs / risks
- Recommendations
- Sources
