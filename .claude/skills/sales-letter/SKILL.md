## Structure

セールスレターの基本構造：
1. ヘッドコピー（キャッチコピー）
2. 現状認識・問題提起
3. 解決策の提示
4. ベネフィット・未来像
5. 実績・証拠
6. 商品内容
7. 価格提示・価格根拠
8. 特典
9. 保証・サポート
10. CTA（注文ボタン）
11. 追伸

## Usage

```
ユーザー: セールスレターを書いて
→ 新版27ブロック構造を使用

ユーザー: ヘッドコピーを考えて

ユーザー: 価格の見せ方を教えて
→ 価格提示パート、衝撃価格提示パートを参照
```

## Guidelines

- 感情を動かす表現を重視
- 価格根拠を論理的に展開
- ベネフィットは具体的に
- CTAは明確に
