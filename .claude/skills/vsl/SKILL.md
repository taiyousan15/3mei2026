# VSL Skill

ビデオセールスレター（VSL）スクリプト作成スキル。

## Knowledge Location

`VSL/` ディレクトリを参照

### 15章構成

1. **第1章：冒頭の大フック＆警告** - 視聴者の注意を一瞬で掴む
2. **第2章：超リアル"Before"ストーリー** - 共感を生む過去の苦悩
3. **第3章：最初の証拠提示（Proof #1）** - 信頼性の構築開始
4. **第4章：ネガティブ訴求・第一波** - 社会情勢・将来不安
5. **第5章：3つの顧客事例A・B・C** - ビフォーアフター
6. **第6章：一瞬のポジティブ訴求・第一波** - 未来イメージ
7. **第7章：さらに深いネガ・第二波** - 騙され経験・リスク
8. **第8章：大量の反論つぶし** - Q&A先出し×10
9. **第9章：追加顧客事例D・E・F** - 多様な背景
10. **第10章：価値観教育** - 常識破壊→新フレーム＆ノウハウ一部公開
11. **第11章：中盤証拠提示（Proof #2）** - 感情揺さぶり
12. **第12章：最終証拠（Proof #3）** - 複数インタビュー形式
13. **第13章：オファー提示** - 商品名・内容・価格＋特典＆限定性
14. **第14章：返金保証・先着枠・最終反論つぶし** - Q&A後半
15. **第15章：クロージング総仕上げ** - 今動かなければ損→行動

## Structure Pattern

VSLの感情曲線：
```
ネガティブ → 希望 → ネガティブ深化 → 解決策 → オファー → 緊急性
```

## Usage

```
ユーザー: VSLの台本を作って
→ 15章構成に沿って作成

ユーザー: 冒頭のフックを考えて
→ 第1章：冒頭の大フック＆警告を参照

ユーザー: 反論つぶしのパートを書いて
→ 第8章、第14章を参照
```

## Guidelines

- 感情の起伏を意識した構成
- 証拠は3段階で提示（Proof #1, #2, #3）
- 顧客事例は多様な背景を用意
- ネガティブとポジティブの波を作る
- クロージングは緊急性を持たせる
