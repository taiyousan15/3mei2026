# Step Mail Skill

ステップメール・メルマガ作成スキル。

## Knowledge Location

`ステップメール/` ディレクトリを参照

### 6つの教育要素（旧版・新版共通）
1. **目的の教育** - 読者の心を掴み、行動を促す熱狂の創造
2. **問題点の教育** - 課題認識の深化
3. **手段の教育** - 解決策の提示
4. **投資の教育** - 価値と対価の理解
5. **信用の教育** - 実績で心を掴み、ファン化を加速
6. **行動の教育** - 具体的アクションへの誘導

### メルマガ執筆技術
- メルマガタイトル奥義（開封率向上）
- メルマガ1通の長さの法則
- 行間とリズムの法則
- リアルタイム感演出
- 予告活用の完全体系

### セールス型テンプレート
- 分岐後即セールス型
- 高額商品セールス型

## Structure

ステップメールの基本構造：
1. 登録直後（ウェルカム）
2. 目的の教育
3. 問題点の教育
4. 手段の教育
5. 投資の教育
6. 信用の教育
7. 行動の教育
8. セールス

## Usage

```
ユーザー: ステップメールを作って
→ 6つの教育要素に沿って構築

ユーザー: メルマガの件名を考えて
→ メルマガタイトル奥義を参照

ユーザー: 日刊メルマガを書いて
→ 日刊メルマガ執筆ナレッジを参照
```

## Guidelines

- 1通1テーマを徹底
- 件名で開封率を最大化
- 本文はリズムと行間を意識
- 次回への期待を煽る
- 読者との距離感を意識
